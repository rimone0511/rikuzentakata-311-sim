# ユーザーへの質問・要確認事項

作業を止めないためにスキップした判断・確認事項の記録。

## データの正確性(フェーズ2までに解決したい)

- [ ] **震災前の標高データ**: 現在の国土地理院タイルは震災後の嵩上げ地形(高田・今泉地区 約+10m)を
      含むため、フェーズ1では2地区を当時の低地(約2.5〜2.8m)に戻す近似補正を入れた
      (tools/fetch_dem.py の KASAAGE_REGIONS)。
      正確な再現には基盤地図情報の旧版DEM(2008年測量など)が必要で、
      ダウンロードには国土地理院サイトのアカウント登録(無料)が必要。
      → ユーザーが登録するか、近似のままでよいか判断してほしい
- [ ] **解体済みランドマークの位置**: 旧市役所・市民会館・市民体育館・旧陸前高田駅は
      現在の地図に存在しないため、概略位置で配置している(assets/landmarks_manual.json の approx:true)。
      震災前の住宅地図・公開資料での位置検証が必要
- [ ] **道路網が震災後のもの**: OSMの現道路網(嵩上げ後の新しい道路を含む)を使用中。
      震災前の街路の再現はフェーズ2で公開資料(震災前の都市計画図など)を基に行いたい
- [ ] **建物配置は手続き生成**: 実際の震災前の建物配置データ(基盤地図情報の建物外周線 旧版)も
      アカウント登録があれば取得できる可能性がある

## 津波再現の数値(フェーズ1では公開報告の代表値を使用)

- [ ] 到達時刻・水位の時系列は「14:46地震 / 15:26頃 防潮堤越流 / 15:33頃 最大水位(浸水深 最大約14〜15m)」
      という公開調査の代表値で近似した。市の検証報告書・痕跡調査(東北地方太平洋沖地震津波合同調査グループ)
      の数値と突き合わせて精密化したい
